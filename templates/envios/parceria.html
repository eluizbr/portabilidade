A CDR-Port (www.cdr-port.net) desenvolveu um sistema de portabilidade numérica com recursos e vantagens interessantes para seus clientes.

Nosso sistema de portabilidade permite ao cliente realizar consultas para determinar à qual operadora um determinado número pertence, retornando ao IP-PABX o código correto da operadora. Com este código, o cliente terá a possibilidade de selecionar o CHIP correto para efetuar a ligação, gerando uma economia real em cada ligação.

Um exemplo simples da utilização do nosso sistema:

  • O cliente liga para o número: 31-9153-0000;
  • O IP-PABX encaminha a solicitação para o nosso sistema;
  • Nosso sistema devolve o número já com o CSP correto da operadora que no qual o número pertence. Em nosso exemplo o retorno seria 55341-31-9153-0000 (Operadora TIM).
  • O IP-PABX do cliente, irá receber essa informação e encaminhar a ligação para o CHIP correto gerando assim economia real para o cliente.

É importate lembrar que essa consulta leva menos de 1 SEGUNDO para ser realizada, não empactando assim no tempo de espera do cliente interno.

Outros destaques de nosso sistema de portabilidade são:

  • Transparente - O processo é automático e transparente para o usuário.
  • Cache - O cliente não é cobrado pelo o mesmo número em um intervalo de até 60 minutos.
  • Painel - Painel WEB com todos os detalhes e estatísticas das consultas realizadas.
  • Rápido - Sistema leva menos de 1 segundo para retornar a consulta.
  • Atualizado - Base de portabilidade atualizada todos os dias.
  • Compatível - Compatível com a maioria das centrais IP do mercado.
